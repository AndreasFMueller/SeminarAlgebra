# Brainstorm and Project Ideas

Die Gamma-Funktion Γ(z) erweitert die Fakultät n! auf beliebige Argumente. Es gilt Γ(n) = (n-1)!. Die Funktion ist als komplexe Funktion durch wenige, natürliche Eigenschaften eindeutig festgelegt.

## Research

- The gamma function is an extension of the factorial function.
- Γ(n) = (n-1)! → The gamma function is offset by one.
- Γ(-0.5) = sqrt(π)
